Description
The network diagram below visualizes a sample subset of telecom subscribers of multiple subscribers within a city. Each node (or bubble) represents a subscriber and the connection between two nodes represent 2 people talking to each other. The size of bubble shows the total number of calls made by the person (i.e. bigger the bubble, more the calls made), while the distance between 2 nodes shows the closeness (i.e. closer bubbles mean that the 2 people talk to each other often). Thus, each of the clusters represent a group of people talking amongst each other. Based on the connections, it is possible to find the direct linkages of one person with all others. Clicking on a bubble highlights all of his/her contacts; moving the mouse-over a bubble highlights relevant details like mobile number, name. It is possible to filter the network to look at call and cluster patterns for a particular day. Additionally, filtering of clusters based on a certain number of subscribers is possible (ex. Enter 5 to filter the network down to clusters having 5 or less number of people in each cluster). With this network visualization, it is easy to view the network of subscribers, identify heavy-usage subscribers or drill-down from large clusters to small ones having a restricted set of individuals calling only amongst themselves.
